<hr /> - для создания полоски которая отделяет страницу на части

ДЛЯ ТАБЛИЦ: 
<table> </table> - таблица
<thead> </thead> - вверх таблицы (должность, компания, дата создания и тд)
<th> </th> как раз таки сами строки сверху таблицы. (<th> Компания </th>
<tbody> </tbody> - тело таблицы, данные таблицы
<td> </td> - как раз таки строки с данными, самого тела таблицы, ячейка данных
<tfoot> </tfoot> - инфа которая будет находится сверху или снизу таблицы
<tr> </tr> - используется для обозначения того, что строчка началась и закончилась. Хочешь сделать несколько строк = <tr> </tr> умножь на количество строк.

ДЛЯ РАБОТЫ С ФОРМАМИ:
<form> </form> - для работы с формами
атрибдуты для form:
action="скрипт который будет указывать форму" (пока инфы нет, в рамках html, но позже его можно связать с например java script. Если в поле вбить /, то при нажатии на кнопку с отправкой данных, нас перекинет на эту же страницу)
<input> - для вписывания данных
несколько атрибутов input: 
type="text" - для вписывания текста.  
id="name" - id для поля с добавлением текста своего. Он работает в связке с label (смотри дальше)
placeholder="текст который будет отображаться на поле с вводом данных, он будет отображаться пока ты не напишешь что-то в это поле" (value="данные, которые уже будут находиться в поле для ввода данных, например 
value="@mail.ru") 
type="email" - поле для вписывания email
type="number" - поле для ввода именно числа
type="password" - поле для ввода пароля (он будет скрытным)
type="color" - поле для выбора цвета
type="date" - поле для выбора даты
type="radio" - кнопочка в виде кружочка, обычно используется во всяких тестах с вариантами ответов
type="submit" - кнопка, при нажатии котором иннфа будет отсылаться куда-то (value="текст который будет отображаться на кнопке, которая отсылает данные") 
<label> </label> - для добавления названия для input (ваше имя) 
атрибуты для label:
for="name"- для того чтобы при нажатии на что-то с определенным id, вас автоматически перенаправляли на строку с написанием текста. (нажал на ваше имя = начал писать текст)
for="email" - для добавления email 
for="text" - для добавления своего текста 
<textarea> </textarea> - для вписывания текста, несколько атрибутов для него:
id="какое-то id чтобы связать его с другой частью кода"
rows="число, обозначающее кол. строчек"
cols="число, обозначающее кол. столбцов"

ЧУТКА ИЗ РАЗРЯДА CSS:
body style=""
style="background" - добавление цвета на задний фон
style="color" - добавление цвета на какой-то элемент (напр h2)
<style> </style> - добавление какого-то стиля самой странице. При добавлении в эти теги какую-то часть страницы, мы обратимся КОНКРЕТНО к этому элементу, например мы хотим обраиться к body. 
Например: 
<style> 
        body {
background: aqua;
        }
    </style>
