css
пример:

body {
background: aqua;
        }

background (background-color - то же самое) - выбор цвета заднего фона
color - для изменения цвета шрифта
font-size: ..px - для определения размера шрифта 
font-weight: ..; - жирность шрифта
font-family:...; - весь стиль шрифта сайта
font: жирность размер семейство; - можно записать все в одну строку
class="название класса"
для class используй .название_класса, а для id используй #название_id
border-radius: ..px; - для скругления фотки
<div> </div> - блочный эелемент, создает свой блок. с помощью этого кода мы создаем блок, к которому потом с помощью css создаём стилизацию. Работает он с классами или id. 
margin: 0; - служит для создания отступов, число - кол. отступов.
font-style: - служит для того, чтобы контроллировать наклонность текста. по умолчанию он стоит на normal. 
text-decoration: - меняет вид текста, декорации так сказать, например нижнее подчеркивание.
text-transform: - меняет регистрв букв в тексте (прописные: uppercase, строчные: lowercase, каждое слово с большой буквы: capitalize, none - отменяет все)
letter-spacing: - создает кол. пикселей которое будет между буквами.
word-spacing: - указывает расстояние между словами. 
.container,  вспомогательная команда для создания блока кода с классом.
div# - вспомогательная команда для создания блока кода с тегом. 
width: в Box Model - свойство которое позволяет уместить блок код в определенное пространство на странице, например, если будет указано 75%, то  этот блок займет 75% страницы. Также можно указывать в пикселях. 
max-width: в Box Model - размер блока будет адаптироваться под устройство, при этом указывается его максимальный размер на странице. 
margin: auto; - перемещает страницу в центр
margin: .px; - создает расстояние между этим бокс-моделем и прочим.
padding-top (-bottom, right, left и тд) - используется для создания внутреннего отступа от текста. 
также можно записывать через padding: 15px; padding: 40px 15px; padding: 25px 20px 30px 40px;
border - создает рамку для этой бокс-модели. 
border-...-style: ; - выбирает стиль рамки (пунктиры, точки, двойная граница и тд).
border-...-color - для выбора цвета рамки
border-...-width - для ширины
.box-model > div { } - это обычный класс, но, там стоит знак >, он нужен для того, чтобы применять стили только к тем, которые лежат на одном уровне. Только дети будут применять эти стили. 
.table td:nth-child(2) - выделяет именно 2 колонку в таблице, чтобы взаимодействовать именно с этой колонкой. 
.table tr:nth-child(even) - свойство, которое дает все четным строчкам опред. свойства. 
.table tr:nth-child(odd) - свойство, которое дает всем нечетным строчка опред. свойства. 
display: ; - свойство, которое определяет тип отображения элемента на веб-странице. (например display: block; делает так, чтобы элемент занимал всю ширину и начинался с новой строки.)
font-family: inherit - inherit нужен для того чтобы элемент унаследовал свойства у других элементов соседей. например чтобы элемент взял такой же шрифт как у остального текста.
position: static; - какой-то объект статично находится в боксе.
position: absolute; - объект не является контентом для какого-то блока.
у position также есть свойства. 
top: ..px; - объект будет находится в стольки то блоках от вверха
left ..px; - от левой стороны
right ..px; - от правой стороны
bottom ..px; - от низа
@media ("условие при котором срабатывает действие") - Конструкция, которая позволяет задавать определенным блокам, которые выполняют условие, определенное значение. 
например: 
@media (max-width: 700px){
    .info {
        text-align: center;
        display:block;
    }
}

То есть, если макс ширина - 700px, то мы описываем набор правил, которые будут применяется на ширине до 700px. display: block; и  align: center; позволят контенту адаптироваться под мобильное устройство, перемещая контент в центр.
